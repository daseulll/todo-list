# Todo 앱
- Java와 Spring Boot로 구현한 할일(Todo)을 관리하는 앱입니다.
--- 



### 사용기술
- Java
- Spring Boot
- Junit5
- h2 database



### API 목록
**회원 기능**
- 회원 가입 API
- 회원 로그인 API
- 회원 로그아웃 API
- 회원 탈퇴 API



**할일 관리 기능**
- 가장 최근 할일(TODO) 1개 조회 API
- 전체 할일 조회 API
- 할일 상태 변경 API 
    - 할 일, 진행중, 완료됨
